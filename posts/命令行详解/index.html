Linux常用命令

进入目录 cd (change directory)
1. cd . 当前目录
2. cd .. 当前目录的父目录

展示文件
1. ls展示当前目录下的非隐藏文件(.开头的文件就是隐藏文件) ls -a 全部展示
2. ls -l 等价于ll 展示详细信息
3. ls -alth (list) 相当于 -a (all) -l (详细) -t (按照文件修改顺序倒叙排列) -h (人类可读模式) 四个命令

移动/重命名文件 mv (move)

删除文件 rm/rm -r(r 是递归参数，把文件及文件下的所有参数都删掉)/rm -rf(f 指强制删除 小心使用) (remove)
* ctrl + c 中断过程

拷贝文件 cp/cp -r(r 同上)

显示变量 echo
1. echo 命令的功能是在显示器上显示一段文字，一般起到一个提示的作用.
2. echo 123 输出123 ；echo 123 > file.txt(> 重定向标志 将输出结果定向到某个文件中; >>则是以追加的方式重定向)

查看(显示)文件内容 cat
* cat file.txt 输出123

导出变量 export
1. export XXX=1
2. echo $XXX
3. 输出1

git系列操作 见git部分

新建目录 mkdir/mkdir -p(p参数 parent 所有的父子组件一次性创建好) (make directory)
* mkdir -p a/b/c/d/e/f

当前目录 pwd (print working directory)

编辑文件 vi 上手有难度 详见 Linux vi深入简出
